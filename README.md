# Python
a blog for my django study

数据库采用MySql

目前已实现功能：

1、用户注册、登陆、上传用户头像

2、文章发表，分类，标签集合

3、评论, 显示用户评论过的文章

4、二维码转换

5、站点缓存

6、完整的测试

7、jQuery, bootstrap

8、markdown支持

9、whoosh搜索

10、logging记录log信息

11、主从数据库(详情可以进入网站http://cblog.xyz)
