# Python
A blog use django

数据库采用MySql

目前已实现功能：

1、用户注册、登陆、上传用户头像

2、文章发表，分类，标签集合

3、评论, 显示用户评论过的文章

4、文件上传、下载，权限控制

5、二维码转换

6、站点缓存

7、完整的测试

8、jQuery, bootstrap

9、markdown支持

10、whoosh搜索

11、logging记录log信息

12、主从数据库(详情可以进入网站http://cblog.xyz)

目前项目已部署在阿里云服务器中，<a href='http://cblog.xyz' target='_blank'>网址</a>为http://cblog.xyz
